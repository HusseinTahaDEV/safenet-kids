# 🛡️ SafeNet Kids
**Next-Generation Digital Safety Education Platform for Children**

SafeNet Kids is an enterprise-grade, interactive educational web application built to teach children and young adults about cybersecurity, digital privacy, and online etiquette through hands-on simulations and gamified quizzes. 

![SafeNet Kids Banner](/public/images/hero.png)

## 🌟 Core Features

- **Interactive Learning Modules**: 
  Step-by-step interactive lessons on Phishing, Passwords, Privacy, and Stranger Danger with AI-generated visual assets.
- **Hands-on Simulators**:
  - **Phishing Detector**: Analyze emails to detect social engineering.
  - **Password Strength Analyzer**: Real-time password cracking estimates and feedback.
  - **Safe Chat Simulator**: Navigate safe responses in simulated online conversations with strangers.
- **Final Knowledge Assessment**: 
  A comprehensive timed quiz system with scoring, XP rewards, and dynamic feedback.
- **Accessibility & UX**: 
  High-fidelity UI scaled perfectly for readability, featuring responsive glassmorphism designs, fluid animations via Framer Motion, and completely localized Arabic content (RTL).

## 🚀 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel CI/CD via GitHub

## 🛠️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HusseinTahaDEV/safenet-kids.git
   cd "SafeNet Kids"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
